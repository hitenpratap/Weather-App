HWeather is a simple Weather reporting app for android.It gives you weather information for selected location. It's a very light weight app(around 62.553 kb only).

Download HWeather View On GitHub

Technologies Used

  • System - Ubuntu 14.04

  • Mobile - Google Nexus 4

  • IDE - Intellij Idea 13

  • Language - Java 1.7

  • Code Management - GitHub

  • Misc - Fragments, Open Weather API, Shared Preferences etc
